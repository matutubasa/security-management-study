📝 模擬試験

本番と同じ形式で実力をチェックしましょう。 60問120分の試験で、あなたの現在の実力を正確に把握できます。

模擬試験について

📊

問題数

60問（四択式）

⏰

試験時間

120分

🎯

合格基準

600点以上（1000点満点）

📈

出題分野

全分野からバランス良く出題

過去の模擬試験結果

まだ模擬試験を受験していません。

初回の模擬試験で現在の実力を確認しましょう！

模擬試験を開始

準備ができたら、下のボタンをクリックして模擬試験を開始してください。

開始前のチェックリスト

  • ✓ 静かな環境で集中できる状態
  • ✓ 120分の時間を確保
  • ✓ インターネット接続が安定
  • ✓ 筆記用具を準備（メモ用）
